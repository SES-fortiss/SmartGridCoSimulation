Schritte zum überspielen auf den Arduino:
1. Arduino IDE herunterladen und installieren http://arduino.cc/en/Main/Software
2. IDE starten, Sketch->Import Library->Add Library
	->Die Bibliothek MotorDriver.zip auswählen(im Repository) -> OK
	http://arduino.cc/en/Guide/Libraries
3. WindradControl.ino öffnen
4. Kompilieren und auf Arduino laden, evtl. noch den COM-Port anpassen (->Tools->Serieller Port)
5. Fertig
